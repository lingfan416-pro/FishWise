# FishWise Backend Baseline

This is a small Express backend for the FishWise MVP.

## Features
- `GET /api/health`
- `GET /api/spots`
- `POST /api/recommendation`
- Local rule-based recommendation logic
- No database
- No authentication
- No external APIs

## Run
```bash
npm install
npm run dev
```

The server runs on `http://localhost:3001` by default.
